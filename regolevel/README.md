## Instalation
```sh
regolith install github.com/llgava/regolith-filters/regolevel
```

## Configuration
```json
{
  "filters": [
    {
      "filter": "regolevel",
      "settings": {
        "levelPath": "./packs/level",
        "worldName": "Spellcraft",
        "ignore": []
      }
    }
  ]
}
```
