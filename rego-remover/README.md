# rego-remover
Remove extra files or objects from the build such as: .bbmode, .ase, Geckolib Animation Artifacts and Aseprite Atlas Configurations.

## Instalation
```sh
regolith install github.com/llgava/regolith-filters/rego-remover
```

## Configuration
```json
{
  "filters": [
    { "filter": "rego-remover" }
  ]
}
```
