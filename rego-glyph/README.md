# rego-glyph
Generates a markdown containing all icon configuration in the `glyph_E1.png` image.<br />
[Click here](./example/) to see an example.

## Instalation
```sh
regolith install github.com/llgava/regolith-filters/rego-glyph
```

## Configuration
```json
{
  "filters": [
    { "filter": "rego-glyph" }
  ]
}
```
